# Wikipedia Research Agent

A simple autonomous research agent built with Python and Wikipedia APIs.

This project demonstrates how to build a beginner-friendly research agent that can search Wikipedia, fetch article summaries, follow internal links, and generate structured research results.

---

## Features

- Search Wikipedia for any topic
- Fetch clean article summaries using Wikipedia REST API
- Extract internal Wikipedia links
- Explore related articles using breadth-first search
- Control search depth and maximum number of pages
- Print readable research results in the terminal

---

## How It Works

The agent follows this workflow:
```text
User Topic
↓
Search Wikipedia
↓
Get Top Articles
↓
Fetch Article Summaries
↓
Extract Internal Links
↓
Explore Related Pages
↓
Print Research Results
